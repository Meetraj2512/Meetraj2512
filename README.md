### 💼 What I've Built

- **🔁 LLM-powered CI automation** — A Claude skill wired into Jenkins that triggers automation builds on every PR, validates results, auto-manages test tags, and routes PRs for review. Saves 5–10 min per PR across the team.
- **🎯 Multi-agent test generation** — An AI system that auto-generates test cases from Jira tickets by orchestrating sub-agents over historical RCA context, existing cases, and PR diffs. Used to author and maintain 400+ test cases.
- **🧹 Intelligent ticket triage** — An n8n-based system that classifies, prioritizes, and deduplicates Sentry monitoring tickets — cutting manual triage effort by 60%.
- **⚡ Automation at scale** — 800+ automated UI/API test cases and CI/CD pipelines that reduced the regression cycle from 12 → 9 hours.
